Rent-A-Car is an open source e-commerce solution built for Wordpress.

== Description ==

Rent-A-Car is open source software that can be used to build online car rental shops quickly and easily. Rent-A-Car is built on top of Wordpress, a world class content management system. Car_Rental is built to be both simple and elegant yet powerful and scalable.

This current version is not complete, we hope to release a workable version soon.

Main Features:

* You can rent your cars out for duration of days;
* You can upload pictures and descriptions of cars
* Simple user management and customer order logs;
* Custom GUI (Graphical User Interface) for car management;


== Installation ==

1. Upload the Rent-A-Car folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Make sure your web host allows scripts like timthumb.php to run on your site.  If they don't, your Rent-A-Car images will be broken.
